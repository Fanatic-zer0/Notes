# CKS Notes

## Overview

This folder contains a deep, exam-focused study guide for the Certified Kubernetes Security Specialist certification.

Start here:

- [study_guide.md](study_guide.md) - comprehensive guide covering cluster hardening, system hardening, workload security, supply chain security, runtime detection, examples, and lab drills.
- [Labs.md](Labs.md) - scenario-based hands-on workbook with timed drills, task prompts, validation steps, and answer keys.

Suggested use:

1. Read the guide once end-to-end to build the security model.
2. Work through the scenario labs without looking at the answers first.
3. Recreate the examples and validation steps in a practice cluster.
4. Revisit the fast command reference and final review sheet before the exam.
